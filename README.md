# Asset Leverage Audit Tool
### billcaskey.com | UPWARD.

A live hot seat tool for Bill Caskey's webinars. Type a volunteer's answers during the session, hit Generate, and the AI instantly produces their personalized Asset Leverage Plan.

---

## Deploy to Vercel (5 minutes)

### Step 1 — Upload to GitHub
1. Create a new GitHub repo called `asset-leverage-audit`
2. Upload all files from this folder into the repo

### Step 2 — Connect to Vercel
1. Go to [vercel.com](https://vercel.com) and log in
2. Click **Add New Project**
3. Import your `asset-leverage-audit` GitHub repo
4. Framework preset: **Create React App**
5. Before deploying, add this environment variable:

| Variable | Value |
|---|---|
| `ANTHROPIC_API_KEY` | Your Anthropic API key |

6. Click **Deploy**

### Step 3 — Optional: Custom Domain
Point a custom domain (e.g. `audit.billcaskey.com`) to the Vercel deployment in your domain settings.

---

## How to Use During a Webinar

1. Open the tool on your screen (share screen with audience)
2. Bring a volunteer into the hot seat
3. Interview them live — type their answers into each field
4. Hit **Generate Asset Leverage Plan**
5. The plan appears on screen — debrief it live with the audience
6. Hit **Print / Save PDF** to give the volunteer a copy
7. Hit **New Hot Seat** to clear and run another

---

## The 11 Intake Fields

1. First Name
2. Role / Title
3. Industry & What They Sell
4. Years in Sales / Business Development
5. Biggest Current Sales Challenge
6. How They Currently Generate Leads
7. Content, IP, or Frameworks They've Created
8. Best Client / Partner Relationships
9. Technology & Tools They Use
10. Biggest Time Wasters Each Week
11. What's Working (Even a Little)

---

## The Report Structure (AI Output)

- **Section 1:** Current Reality Snapshot
- **Section 2:** Asset Inventory (Knowledge / Relationship / Content / Technology / Time)
- **Section 3:** Top 3 Underdeployed Assets (with gap analysis)
- **Section 4:** 90-Day Activation Sequence (Month 1/2/3)
- **Section 5:** The Bill Caskey Challenge (direct mindset question)
